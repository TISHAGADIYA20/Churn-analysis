# Customer Churn Analysis & Prediction

## Overview

This project presents an end-to-end customer churn analytics solution that combines SQL, Power BI, Python, and Excel to analyze customer behavior, identify churn drivers, and predict customers at risk of leaving. The project demonstrates the complete analytics workflow, from data cleaning and transformation to interactive dashboards and machine learning.

---

## Objectives

- Analyze customer churn trends and key business KPIs.
- Identify factors influencing customer churn.
- Build an interactive Power BI dashboard.
- Develop a machine learning model to predict churn for new customers.
